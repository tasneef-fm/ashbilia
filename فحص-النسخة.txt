فحص نسخة وردة أشبيليا — GitHub Pages
=====================================

تم التحقق من:
- وجود index.html في جذر المشروع.
- جميع مسارات الصور وCSS وJavaScript نسبية وتعمل من مستودع فرعي على GitHub Pages.
- صحة بناء JavaScript في app.js وsupabase-api.js.
- صحة بناء SQL نحويًا لملفي schema.sql وseed-demo.sql.
- وجود GitHub Actions للنشر التلقائي من فرع main.
- عدم تضمين server.js أو SQLite في نسخة GitHub.
- عدم تضمين كلمات مرور أو مفاتيح خاصة.

لا يمكن اختبار الاتصال الفعلي أو تسجيل الدخول قبل إدخال رابط مشروع Supabase
والمفتاح العام وإنشاء حساب الإدارة، لأن هذه البيانات تخص حساب مالك التطبيق.
